# Chat app for TFS
Приложение чат, разделение на каналы, с возможностью отправлять текст и фото, смена темы приложения, экран профиля 

## Technologies

- UIKit
- Code autolayout
- Core dara
- Fetched Result Controller
- FileManager
- Firebase
- SOA/MVC architecture
- Unit tests
- UI tests
